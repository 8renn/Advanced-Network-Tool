# Advanced Network Tool (A.N.T.) v1.0

**Portable & Installable Edition**

Advanced Network Tool is a Windows desktop diagnostics suite designed for IT professionals, network engineers, and support teams. It combines several practical troubleshooting tools in a single interface, with sidebar-based navigation and unified reporting.

## Included Tools

- **IP Scanner** - Discover active devices on your local network
- **MTR** - Run continuous traceroute tests with latency and loss statistics
- **Traceroute** - Map the network path to a destination host or IP
- **SIP ALG Detector** - Check whether a router/firewall is modifying SIP/VoIP traffic
- **System Info** - Collect local network details, public IP, and speed test results
- **Full Report** - Compile collected results into one exportable text report
- **Settings** - adjust layout, themes, and default tables.

## Basic Usage

1. Launch `AdvancedNetworkTool.exe`
2. Use the left sidebar to switch between tools
3. Hover the red info dot beside each page title for contextual help
4. Open **Full Report** to generate and export a consolidated report

## Requirements

- Windows 10 or later (64-bit)
- No Python installation required for the portable executable
- Administrator privileges are recommended for best diagnostics coverage
- Active network connection for scans and online tests

## Portable & Installable Edition

This is a portable and installable application, You will have the option upon launch. 
You can run it from a local folder, USB drive, or network share.

## Technology Stack

Built with **Python**, **PySide6**, and the **Windows ICMP API**.
