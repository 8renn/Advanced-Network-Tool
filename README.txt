===============================================================================
  Advanced Network Tool (A.N.T.) v1.0
  Portable Edition
===============================================================================

WHAT IS THIS?
  A comprehensive network diagnostics suite for IT professionals.

TOOLS INCLUDED:
  - IP Scanner        Discover devices on your local network
  - MTR               Continuous traceroute with latency statistics
  - Traceroute        Map network path to any destination
  - SIP ALG Detector  Check if your router modifies SIP/VoIP traffic
  - System Info       Network config, public IP, and speed tests
  - Full Report       Export all results to a text file

HOW TO USE:
  1. Double-click AdvancedNetworkTool.exe to launch
  2. Use the sidebar to navigate between tools
  3. Hover the red info dot next to each page title for help
  4. Use Full Report to compile and export all results

REQUIREMENTS:
  - Windows 10 or later (64-bit)
  - No Python installation needed
  - Administrator privileges recommended
  - Network connection for scanning and testing

PORTABLE:
  This is a portable application. No installation required.
  Run from any folder, USB drive, or network share.

===============================================================================
  Built with Python, PySide6, and the Windows ICMP API
